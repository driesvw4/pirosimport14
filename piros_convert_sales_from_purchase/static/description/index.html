Generate Sale Order from Purchase Order

This module give feature to create sale order to your customer for product you want sale them based on Purchase request (Only if not in Done Or Cancel State) from your vendor.

Key Features:

  • You can create Multiple Sale Order directly from Purchase Order in one go.
  • Allow Product Quantity & Price modification at time of Order Generate.
  • Generate Sale Order to the supplier easily with one-click wizard on Based on Purchase Order Line Data.

How To Use


Create Purchase Order

After installing this module when you create purchase order the "Create Sale Order" button will appear in Purchase Order.


Convert Sale Order

When you click on "Create Sale Order" button the new wizard will Open with current purchase order line .


You can see that new order is created in sale order.
